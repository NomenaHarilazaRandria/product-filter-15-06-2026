# Product Filter App

A responsive product filtering application built with Vanilla JavaScript, HTML, and CSS.

This project was created as part of my Front-End learning journey to practice working with arrays of objects, filtering data, and dynamic DOM rendering.

## Features

- Live product search
- Product suggestions while typing
- Category filtering
- Maximum price filtering
- Multiple filters combined together
- Responsive design (mobile, tablet, desktop)
- Dynamic product rendering
- Keyword matching with highlighted search terms

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Concepts Practiced

- Arrays of objects
- Array methods:
  - `filter()`
  - `map()`
  - `some()`
- DOM manipulation
- Event handling
- Responsive design
- Search and filtering logic

## How It Works

Users can search for products using keywords, select a category, and define a maximum price.

The application filters products dynamically by combining:

Search input
Category selection
Maximum price selection

Only products matching all selected criteria are displayed.
